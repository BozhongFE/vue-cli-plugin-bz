# vue-cli-plugin-bz

对 vue-cli@3.x 模板进行二次定制化，使之符合团队模板开发

## 安装

```
$ vue add bz
```

### 支持类型

+ Router
+ Router History Mode
+ Less
+ Vuex
+ Redirected 地址重定向
+ Source 打包到 source 版本库

### 步骤

+ 使用 `vue-cli@3.x` 创建新项目
+ 进入项目目录，执行 `vue add bz` 命令进行插件安装并模板个性化

### 分享图相关

分享图片请放于 `/public/img/xxxx`, 例如 `/public/img/share.png`，build 过程会自动复制到构建目录
